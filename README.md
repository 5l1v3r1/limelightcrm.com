# How to install?

```
$ git clone git@github.com:mahendrakalkura/limelightcrm.com.git
$ cd limelightcrm.com
$ cp settings.php.sample settings.php
```

# How to use?

```
$ cd limelightcrm.com
$ composer install
$ php main.php --list-products
$ php main.php --test-products
$ php main.php --list-gateways
$ php main.php --test-gateways
$ php main.php --list-campaigns
$ php main.php --test-campaigns
$ php main.php --test-orders
```
